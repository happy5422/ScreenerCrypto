# ScreenerCrypto

Универсальный скринер криптовалютных пар с поддержкой множества стратегий и бирж.

## 🚀 Особенности

Проект включает в себя ряд специализированных скринеров для поиска торговых возможностей в реальном времени:

- **Scalping**: Отслеживание всплесков объема, CVD и дисбаланса стакана (Order Book Imbalance).
- **Bot Detector**: Обнаружение активности маркет-мейкеров и алгоритмов (MM Algo, Wash Trading, Spoofing).
- **Grid Zones**: Поиск монет в узких горизонтальных диапазонах для сеточной торговли.
- **Midterm**: Анализ открытого интереса (OI), трендов объема и дивергенции CVD на таймфреймах 1h-4h.
- **Alpha**: Поиск аномалий на низколиквидных рынках (аномальный фандинг, спреды относительно лидеров рынка).
- **Iceberg Orders**: Детекция скрытых заявок в стакане.
- **Arbitrage**: Поиск межбиржевых арбитражных возможностей.
- **Basis**: Мониторинг базиса (премии) между бессрочными и квартальными фьючерсами.
- **Liquidations**: Отслеживание крупных ликвидаций.
- **MEXC 0% Fee**: Поиск пар с нулевой комиссией на бирже MEXC.
- **Sentiment**: Анализ социальных настроений.
- **Listings**: Мониторинг новых листингов.

## 🛠 Технологический стек

### Backend
- **FastAPI**: Современный, производительный веб-фреймворк на Python.
- **CCXT**: Библиотека для интеграции с криптобиржами.
- **Asyncio**: Асинхронная обработка данных для высокой скорости работы.

### Frontend
- **React**: Библиотека для создания пользовательских интерфейсов.
- **Framer Motion**: Продвинутая анимация интерфейса.
- **Lucide React**: Современный набор иконок.
- **Tailwind CSS**: Гибкая стилизация компонентов.

## 📂 Структура проекта

- `/backend`: Серверная часть, логика сбора и анализа данных.
  - `/app/routes`: API эндпоинты для скринеров.
  - `/app/services`: Сервисы для взаимодействия с биржами и расчет индикаторов.
- `/frontend`: Клиентская часть, визуализация сигналов.
  - `/app/src/components/screener`: Компоненты отдельных скринеров.
  - `/app/src/routes`: Основные страницы приложения.

## ⚙️ Установка и запуск

### Backend
1. Перейдите в папку `backend`.
2. Установите зависимости (python 3.9+):
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите сервер:
   ```bash
   uvicorn app.main:app --reload
   ```

### Frontend
1. Перейдите в папку `frontend/app`.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите приложение:
   ```bash
   npm run dev
   ```
